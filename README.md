# 此刻点评
&ensp;&ensp;&ensp;此刻点评是一个前后端分离项目，类似于大众点评。项目主要实现了短信登陆、优惠卷秒杀、达人探店、好友关注等功能，平台可以给商家增加曝光度来提升流量，为用户提供查看附件商铺、一键签到等功能。
